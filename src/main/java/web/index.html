Socket地址输入框

② 端口号输入框

用于输入服务器端口号，如 80 用于网页访问，22 用于 SSH 等。

Socket端口输入框

③ 连接 / 断开 按钮

用于建立或断开与服务器的 Socket 连接，便于测试连接是否正常。

连接断开按钮

④ 发送消息输入框

输入要发送给服务器的文本内容或指令，例如简单字符串或协议命令。

消息文本输入框

⑤ 发送按钮

点击后，会把“发送消息输入框”中的内容通过当前的 Socket 连接发送出去。

发送Socket按钮
